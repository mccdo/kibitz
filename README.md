kibitz
======

A message centric grid computing environment


Dependencies
------------

ZeroMQ 
```
git clone ssh://git@github.com/zeromq/zeromq-x.git
cd zeromq-x
./autogen.sh
./configure --with-pgm
make 
sudo make install
```
